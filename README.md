# Kademlia-Rust

This is a Rust implementation of the Kademlia distributed hash table (DHT) protocol. 

## Features

- [x] Node ID generation
- [x] Routing table
- [x] Bucket
- [x] local store
- [x] Value store
- [x] xor distance

## Usage

to run the project, use the following command:

```bash
cargo run
```

to run the demo, use the following command:

```bash
cargo run --bin demo
```
